Software Engineering & Data Science Projects

A curated selection of my technical projects, demonstrating my ability to build full-stack applications and integrate machine learning models.

Cologne Recommendation Engine

Problem: Fragrance shopping online is difficult without smelling the product. I wanted to build a system that recommends new colognes based on previously liked scent profiles.

Solution: A full-stack "Spotify for Colognes" application built to provide personalized fragrance recommendations. Developed the frontend using React and the backend with FastAPI. Implemented data science techniques and content-based filtering to match users with their ideal scent profiles based on notes and accords.

FitFuel Mobile App

Problem: Existing nutrition tracking applications are often cluttered with unnecessary features, making simple macro-tracking cumbersome for weightlifting and fitness enthusiasts.

Solution: Developed "FitFuel", a comprehensive mobile nutrition tracking application using React Native and Expo. The app focuses on a seamless user experience, cross-platform compatibility, and intuitive health data management, allowing users to quickly log meals and track nutritional goals.

NLP Fake News Classifier

Overview: Engineered a robust Natural Language Processing pipeline as part of a collaborative group initiative. Leveraged a BERT machine learning model to accurately classify text data and detect misinformation within news datasets, highlighting the intersection of data science and media literacy.
